**Overview**

FlightDash for Planes is a modern full-screen EdgeTX widget designed specifically for fixed-wing aircraft. It combines key telemetry, flight status, battery monitoring, control visualization, and pilot-focused flight information into a clean aviation-inspired dashboard.

The widget was built to provide an at-a-glance flight display while remaining lightweight and responsive on EdgeTX radios. It features animated aircraft roll indication, dual gauge instrumentation, segmented telemetry bars, trim visualization, transmitter battery monitoring, and configurable telemetry sources.

FlightDash is designed for pilots who want a highly visual and easy-to-read telemetry interface without cluttering the screen with unnecessary information.

**Features**
- Full-screen aviation-style dashboard layout
- Dual large instrument gauges
- Animated aircraft roll indicator using aileron input
- Receiver battery analog gauge
- Flight timer display
- Flight mode display
- Motor on/off status indicator
- Transmitter battery monitor
- Live telemetry bars:
  - RSSI / 1RSS
  - RQly
  - Throttle position
- Real-time trim position indicators
- Date and time display
- Altitude telemetry support
- Maximum altitude tracking
- Configurable telemetry sources through EdgeTX widget options
- Lightweight and optimized for EdgeTX 2.11+
- Designed specifically for fixed-wing aircraft
- Smooth animated aircraft bank-angle visualization
- Modern glass-cockpit inspired interface
- Clean segmented bar graphics and gauge styling
